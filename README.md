### Kefka P4Overlay for PC
FF14 絕妖星P4判定輔助工具

## 使用方式
1. 點擊右側Release下載 Kefka P4.exe
2. 直接執行，無需安裝任何額外軟體
3. 建議 FF14 使用「無邊框視窗」模式，讓 overlay 可以顯示在遊戲上方
4. 右上齒輪內可設定
5. 三角形可最小化視窗
6. 設定內可調整透明度、左右翻轉、解除鎖定視窗後可拖動視窗邊緣調整視窗大小

## 功能
Alpha 模式：問答型，只要依看到的順序按下按鈕（GC1/水火1/GC2/水火2/）
Beta 模式：舊的版本移植
支援中文 / 日本語 / English

## 注意事項
首次執行可能出現 Windows SmartScreen 警告
點「更多資訊」→「仍要執行」即可
這是正常現象，程式本身不含任何惡意程式碼

### Kefka P4Overlay for PC
A jassistance overlay for FF14 DMU kefka P4

## How to Use
1. Download Kefka P4.exe
2. Run it directly — no installation or additional software required
3. Recommended: set FF14 to Borderless Window mode so the overlay appears on top of the game
4. Click the gear icon in the top-right to open the settings.
5. the triangle button to minimize the window.
In the settings, you can adjust the opacity, flip the window horizontally, and unlock the window to resize it by dragging its edges.

## Features
Alpha mode: Press the buttons in order, and display the complete solution sequence on the left side (GC1 / Entropy・Fluid1 / GC2 / Entropy・Fluid2 )
Beta mode: Old ver.
Supports 中文 / 日本語 / English

## Notes
Windows SmartScreen may show a warning on first launch
Click "More info" → "Run anyway" to proceed
This is normal behavior for unsigned applications — the program contains no malicious code

### Kefka P4 Overlay for PC
FF14 絶妖星乱舞 P4 判定支援ツール

## 使い方
1. 右側のReleaseクリック　Kefka P4.exe をダウンロード
2. インストール不要。そのまま実行できます
3. FF14 は「ボーダーレスウィンドウ」モードでの使用を推奨します（オーバーレイをゲーム上に表示できます）
4. 右上の歯車から設定できます。
5. 三角アイコンでウィンドウを最小化できます。
6. 設定画面では透明度や左右反転を変更できます。また、ウィンドウのロックを解除すると、ウィンドウの端をドラッグしてサイズを変更できます。

## 機能
Alpha モード：表示された順番にボタンを押すだけで、左側に最終的な処理順を表示します
Beta モード：旧バージョンを移植したモード
日本語 / English / 中文 対応

## 注意事項
初回起動時に Windows SmartScreen の警告が表示される場合があります
「詳細情報」→「実行」を選択すると起動できます
署名のないアプリのため表示されるものであり、悪意のあるプログラムではありません
