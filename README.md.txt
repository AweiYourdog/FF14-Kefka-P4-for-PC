Kefka P4 Overlay for PC
FF14 絶妖星乱舞 P4 判定支援ツール

使い方

Kefka P4.exe をダウンロード
インストール不要。そのまま実行できます
FF14 は「ボーダーレスウィンドウ」モードでの使用を推奨します（オーバーレイをゲーム上に表示できます）
右上の歯車，設定できます。

機能
Alpha モード：表示された順番にボタンを押すだけで、左側に最終的な処理順を表示します
Beta モード：旧バージョンを移植したモード
日本語 / English / 中文 対応
透明度、ウィンドウサイズ、左右反転を調整可能
注意事項
初回起動時に Windows SmartScreen の警告が表示される場合があります
「詳細情報」→「実行」を選択すると起動できます
署名のないアプリのため表示されるものであり、悪意のあるプログラムではありません
